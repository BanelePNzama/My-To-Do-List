# Advanced To-Do List App

A dynamic, interactive To-Do List web application built with **HTML, CSS, and JavaScript**. This project demonstrates DOM manipulation, event handling, and localStorage usage.

---

## **Features**
- Add tasks with a **priority** (High, Medium, Low)
- Mark tasks as **completed/uncompleted** by clicking
- Delete tasks individually
- **Filter tasks** by All, Completed, or priority
- Tasks are stored in **localStorage** to persist across browser sessions
- Clean, responsive UI with visual priority indicators
